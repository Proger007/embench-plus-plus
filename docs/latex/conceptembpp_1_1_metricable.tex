\doxysubsection{Detailed Description}
Concept to define a metric for benchmarking. 

The {\ttfamily Metricable} concept ensures that the type passed as a Metric implements the {\ttfamily get\+\_\+metric\+\_\+start()}, {\ttfamily get\+\_\+metric\+\_\+end()} and {\ttfamily setup()} methods.
\begin{DoxyItemize}
\item {\ttfamily get\+\_\+metric\+\_\+start()} should return a 32-\/bit unsigned integer representing the metric start value (e.\+g., cycle count).
\item {\ttfamily get\+\_\+metric\+\_\+end()} should return a 32-\/bit unsigned integer representing the metric end value (e.\+g., cycle count -\/ 1).
\item {\ttfamily setup()} should initialize or reset the metric system before the benchmark starts. 
\end{DoxyItemize}
