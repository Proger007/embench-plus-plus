\doxysubsection{Detailed Description}
Concept to define a metric for benchmarking. 

The {\ttfamily Metricable} concept ensures that the type passed as a Metric implements both the {\ttfamily get\+\_\+metric()} and {\ttfamily setup()} methods.
\begin{DoxyItemize}
\item {\ttfamily get\+\_\+metric()} should return a 32-\/bit unsigned integer representing the metric value (e.\+g., cycle count).
\item {\ttfamily setup()} should initialize or reset the metric system before the benchmark starts. 
\end{DoxyItemize}
