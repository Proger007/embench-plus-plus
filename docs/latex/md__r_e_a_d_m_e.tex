{\bfseries{embench++}} is a modern, modular, and hardware-\/independent benchmarking framework for embedded systems, designed with C++20 for maximum flexibility and zero-\/runtime overhead.

This framework enables developers to evaluate performance-\/critical functions across various environments and configurations. It supports plugin-\/like extensions for metric collection, data processing, and result output — allowing benchmarking on platforms ranging from bare-\/metal microcontrollers to more complex setups.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Modular architecture}} with clear separation of concerns
\item {\bfseries{Header-\/only core}} with optional modules (metrics, processors, printers)
\item {\bfseries{Zero dynamic allocation}} for real-\/time embedded compatibility
\item {\bfseries{C++20 concepts}} for compile-\/time validation
\item {\bfseries{Optional Docker-\/based development setup}} for fast onboarding and reproducibility
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Repository Structure}{Repository Structure}}\label{md__r_e_a_d_m_e_autotoc_md3}
