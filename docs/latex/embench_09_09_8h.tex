\doxysubsection{Detailed Description}
Header file defining the benchmarking framework for embedded systems. 

This header provides the {\ttfamily Benchmark} class template for running benchmarks on functions, with minimal overhead using a metric system. The framework uses template metaprogramming to inline the function under test (FUT) and optimize the hot path.
