class embpp\+::\+Benchmark$<$ N, Metric, FUT $>$}
Class template for benchmarking functions. 

The {\ttfamily \doxylink{classembpp_1_1_benchmark}{Benchmark}} class runs a function under test (FUT) multiple times while measuring its performance using the provided metric. The metric setup is done before the benchmark starts, and the results are stored in an array.


\begin{DoxyTemplParams}{Template Parameters}
{\em N} & The number of iterations for the benchmark. \\
\hline
{\em Metric} & A type satisfying the {\ttfamily \doxylink{conceptembpp_1_1_metricable}{Metricable}} concept. \\
\hline
{\em FUT} & The function under test (FUT), passed as a template parameter. \\

Runs the benchmark for the specified function. 

The {\ttfamily \doxylink{classembpp_1_1_benchmark_a00eb7f7a3bb884a0e38a01c25be76683}{run()}} method first calls the {\ttfamily setup()} method of the metric to initialize the metric system. It then executes the function under test (FUT) {\ttfamily N} times, recording the start and end metric values around each execution to calculate the duration.

The results are stored in the {\ttfamily results} array.
